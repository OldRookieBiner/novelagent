## 需求 1：章节正文结尾去除数字

### 问题描述

LLM 生成章节正文时，末尾会自动追加上字数统计，如：

```
...故事到此结束。

3247
```

### 解决方案

在后端 `generate_chapter_content_node` 函数中，对 LLM 输出内容进行后处理，移除结尾的纯数字。

### 实现位置

- 文件：`backend/app/agents/nodes/chapter_generation.py`
- 函数：`generate_chapter_content_node`

### 实现细节

```python
import re

async def generate_chapter_content_node(state: NovelState) -> NovelState:
    # ... 现有代码 ...

    # 调用 LLM 生成内容
    content = await llm.chat([{"role": "user", "content": prompt}])

    # 后处理：移除结尾的纯数字（可能是 LLM 自动添加的字数）
    content = re.sub(r'\n*\d+\s*$', '', content.strip())

    # 计算字数
    word_count = len(content)

    # ... 后续代码 ...
```

### 测试用例

| 输入 | 预期输出 |
|------|----------|
| `"正文内容\n\n3247"` | `"正文内容"` |
| `"正文内容\n3247"` | `"正文内容"` |
| `"正文内容3247"` | `"正文内容3247"`（数字在段落中不处理） |
| `"正文内容"` | `"正文内容"`（无数字结尾不处理） |

```tsx

import { toast } from "sonner"

export function useToastError() {
  const showError = (message: string, description?: string) => {
    toast.error(message, {
      description,
    })
  }

  return { showError }
}
```

function AddModelDialog({ open, onOpenChange, onSuccess }) {
  const [provider, setProvider] = useState('deepseek')
  const [showModelList, setShowModelList] = useState(false)
  const [availableModels, setAvailableModels] = useState([])
  const [selectedModels, setSelectedModels] = useState<string[]>([])

  const providerConfig = PRESET_PROVIDERS[provider]

  // 监听 provider 变化，自动填充 base_url
  useEffect(() => {
    if (provider !== 'custom') {
      setBaseUrl(providerConfig.base_url)
    }

  const handleFetchModels = async () => {

        base_url: baseUrl,
        api_key: apiKey
      })
      setAvailableModels(result.models)

      setAvailableModels([])

  }

  return (
    <Dialog open={open} onOpenChange={onOpenChange}>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>添加模型</DialogTitle>

          {/* 提供商选择 */}
          <div>
            <Label>提供商</Label>
            <Select value={provider} onValueChange={setProvider}>
              <SelectTrigger>
                <SelectValue />
              </SelectTrigger>
              <SelectContent>
                <SelectItem value="deepseek">DeepSeek</SelectItem>
                <SelectItem value="baidu">百度千帆</SelectItem>
                <SelectItem value="volcengine">火山方舟</SelectItem>
                <SelectItem value="unicom">联通云</SelectItem>
                <SelectItem value="custom">自定义</SelectItem>
              </SelectContent>
            </Select>
          </div>

          {/* 显示名称 */}
          <div>
            <Label>显示名称</Label>
            <Input value={name} onChange={e => setName(e.target.value)} />
          </div>

          {/* API 地址 */}
          <div>
            <Label>API 地址</Label>
            <Input
              value={baseUrl}
              onChange={e => setBaseUrl(e.target.value)}
              disabled={provider !== 'custom'}
            />
          </div>

          {/* API Key */}
          <div>
            <Label>API Key</Label>
            <Input type="password" value={apiKey} onChange={e => setApiKey(e.target.value)} />
          </div>

          {/* Coding Plan：获取模型按钮 */}
          {providerConfig?.provider_type === 'coding_plan' && (
            <Button onClick={handleFetchModels} disabled={!apiKey}>
              获取可用模型
            </Button>
          )}

          {/* 单模型：模型名称输入 */}
          {providerConfig?.provider_type === 'single' && (
            <div>
              <Label>模型名称</Label>
              <Input value={modelName} onChange={e => setModelName(e.target.value)} />
            </div>
          )}

          {/* Coding Plan：模型选择列表 */}
          {showModelList && (
            <div>
              <Label>选择模型</Label>
              {availableModels.length > 0 ? (
                <div className="border rounded-lg divide-y">
                  {availableModels.map(model => (
                    <label key={model.id} className="flex items-center p-2">
                      <Checkbox
                        checked={selectedModels.includes(model.id)}

                            setSelectedModels(selectedModels.filter(id => id !== model.id))
                          }

                      />
                      <span className="ml-2">{model.name}</span>
                    </label>
                  ))}
                </div>
              ) : (
                <div className="text-muted-foreground text-sm">
                  API 不支持自动获取模型列表，请手动输入模型名称（每行一个）

## 风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 部分 Coding Plan API 不支持模型列表查询 | 用户无法自动获取模型 | 提供手动输入模型的备选方案 |
| 迁移数据时现有配置丢失 | 用户需重新配置 | 迁移脚本自动设置 provider_type 为 single |
| 多模型选择导致 UI 复杂 | 用户体验下降 | 采用分组折叠设计，保持简洁 |

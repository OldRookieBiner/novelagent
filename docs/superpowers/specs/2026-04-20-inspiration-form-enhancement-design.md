# 灵感采集表单优化设计

## Overview

优化灵感采集步骤，新增 4 个配置项，改进 UI 交互体验。

## Changes

### 1. 新增配置项

| 配置项 | 必填 | UI 样式 | 选项 |
|--------|------|---------|------|
| 目标读者 | ✅ | 卡片选择 | 男频、女频 |
| 每章字数 | ✅ | 下拉框 | 1500-2000/2000-2500/2500-3000/3000-5000/自定义 |
| 叙事视角 | ❌ | 标签选择 | 第一人称、第三人称 |
| 金手指 | ❌ | 标签选择 | 系统流/空间流/重生流/穿越流/签到流/无金手指/自定义 |

### 2. UI 样式调整

| 配置项 | 原样式 | 新样式 |
|--------|--------|--------|
| 小说类型 | 下拉框 | 卡片选择（带图标） |
| 目标读者 | 新增 | 卡片选择（独占一行） |
| 核心主题 | 下拉框 | 标签选择 |
| 世界观设定 | 下拉框 | 标签选择 |
| 主角设定 | 下拉框 | 标签选择 |
| 金手指 | 新增 | 标签选择 |
| 叙事视角 | 新增 | 标签选择 |
| 风格偏好 | 下拉框 | 标签选择 |

## Configuration Items Detail

### 必填项（4项）

| 配置项 | UI | 选项 |
|--------|-----|------|
| 目标读者 | 卡片选择（独占一行） | 男频、女频 |
| 小说类型 | 卡片选择（6选1） | 玄幻、科幻、都市、言情、悬疑、历史 |
| 小说篇幅 | 下拉框 | 短篇(10-20章)、中篇(30-50章)、长篇(50-100章)、超长篇(100+章)、自定义 |
| 目标字数 | 下拉框 | 50万、100万、200万、300万、500万、自定义 |
| 每章字数 | 下拉框 | 1500-2000字、2000-2500字(标准·番茄推荐)、2500-3000字(中章·七猫推荐)、3000-5000字、自定义 |

### 选填项（6项）

| 配置项 | UI | 选项 |
|--------|-----|------|
| 叙事视角 | 标签选择 | 第一人称、第三人称 |
| 核心主题 | 标签选择 | 复仇、成长、逆袭、爱情、探险、权谋 |
| 世界观设定 | 标签选择 | 修仙体系、魔法世界、赛博朋克、现代社会、古代王朝、自定义 |
| 主角设定 | 标签选择 | 少年天才、穿越者、重生者、草根逆袭、普通人、自定义 |
| 金手指设定 | 标签选择 | 系统流、空间流、重生流、穿越流、签到流、无金手指、自定义 |
| 风格偏好 | 标签选择 | 轻松幽默、热血激昂、细腻唯美、暗黑深沉、紧张刺激 |

## Data Schema

### InspirationData 新增字段

```typescript
interface InspirationData {
  // 现有字段
  novelType: string
  novelLength: string
  customChapterCount?: number
  targetWords: string
  customTargetWords?: number
  coreTheme: string
  worldSetting?: string
  customWorldSetting?: string
  protagonist?: string
  customProtagonist?: string
  stylePreference?: string

  // 新增字段
  targetReader: string        // 'male' | 'female'
  wordsPerChapter: string     // 每章字数
  customWordsPerChapter?: number
  narrative: string           // 'first' | 'third'
  goldFinger?: string         // 金手指类型
  customGoldFinger?: string
}
```

## Success Criteria

1. 目标读者卡片独占一行，样式突出
2. 小说类型使用卡片选择，带图标
3. 必填项（篇幅、字数、每章字数）使用下拉框
4. 进阶设定全部使用标签选择，每种独占一行
5. 所有新增字段正确保存到后端
6. 灵感模板正确包含新增字段
7. 表单验证正常工作
